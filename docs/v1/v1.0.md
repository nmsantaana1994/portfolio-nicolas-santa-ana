# V1.0 — Primera versión funcional

## Objetivo

Publicar un portfolio profesional, sobrio y mantenible para presentar el perfil de Full Stack PHP/Laravel Developer.

## Funcionalidades entregadas

- Hero con posicionamiento profesional y llamadas a la acción.
- Secciones Sobre mí, Foco de valor y Stack técnico.
- Experiencia resumida y orientada a portfolio.
- Casos de proyectos sin información sensible.
- Descarga segura del CV cuando el PDF está disponible.
- Contacto mediante email, LinkedIn y GitHub.
- Navegación por anclas y menú mobile.
- Diseño responsive con Tailwind CSS 4.
- SEO básico, Open Graph, Twitter Card y JSON-LD `Person`.
- Contenido centralizado en `config/portafolio.php`.
- Tests de home, contenido, metadatos, enlaces y CV.

## Arquitectura

La aplicación se mantuvo intencionalmente simple:

```text
Ruta pública /
    → carga config/portafolio.php
    → renderiza un layout Blade
    → compone parciales por sección
    → carga CSS y JavaScript mediante Vite
```

## Decisiones

- Blade en lugar de un framework SPA.
- Tailwind ya incluido en el proyecto, sin Bootstrap adicional.
- JavaScript nativo para el menú móvil.
- Sin persistencia ni autenticación.
- CV servido como archivo público y validado antes de enlazarlo.

## Resultado

Primera versión funcional, desplegable y preparada para presentar el perfil profesional completo en una única página.
