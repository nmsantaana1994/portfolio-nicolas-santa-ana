## Alcance compartido

- Laravel 13, PHP 8.4, Blade y Tailwind CSS 4.
- Página pública responsive y mobile-first.
- Contenido editable centralizado en `config/portafolio.php`.
- Sin login, base de datos, panel administrativo ni formularios persistentes.
- Deploy en Railway.
